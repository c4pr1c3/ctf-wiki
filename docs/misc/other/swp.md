vim swap file
===

!!! 知识点
    在编辑文件的过程中，Vim将会在当前目录中自动生成一个以.swp结尾的临时交换文件，用于备份缓冲区中的内容，以便在意外退出时可以恢复之前编辑的内容。

- 题目：[ctfx-problems-2016/forensics/crash-50](https://github.com/ctf-x/ctfx-problems-2016/blob/master/forensics/crash-50/writeup.txt)
